# Monigo UI

Embeddable customer-portal UI packages for web (React, Vue, Svelte) and Flutter.

## Packages

- [`@monigo/tokens`](./packages/tokens) — design tokens and theming helpers.
- [`@monigo/portal-core`](./packages/portal-core) — typed API client and state stores.

Framework packages (`@monigo/react`, `@monigo/vue`, `@monigo/svelte`) and the Flutter package arrive in subsequent milestones. See `/Users/akachukwuokafor/Code/Monigo/docs/superpowers/specs/2026-04-18-monigo-ui-packages-design.md`.

## Development

```bash
pnpm install
pnpm build
pnpm test
```

Requires Node 20+ and pnpm 9+.
